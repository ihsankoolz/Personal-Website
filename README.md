# ihsan-portfolio

Personal portfolio website for Ihsan Alfian — built with React, Vite, TypeScript, and TailwindCSS.

## Stack

- React 18 + TypeScript
- Vite 6
- TailwindCSS 4
- Motion (Framer Motion)
- Anime.js
- Recharts

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
